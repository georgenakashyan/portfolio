--shadow-glow-sm: 0 0 8px var(--glow-primary);

  0 0 20px var(--glow-primary),

<task type="auto">
  <name>Task 3: Add inline link base styles</name>
  <files>app/globals.css</files>
  <action>
Add a rule for inline links that aren't in MDX content and don't have explicit classes. Add this after the `.mdx-content` styles section:

```css
/* =========================
   INLINE LINK STYLES
   Ensures links are distinguishable from text per WCAG 1.4.1
   Navigation links excluded (positioned context provides distinction)
   ========================= */
a[href]:not([class]):not(nav a) {
  color: var(--accent-primary);
  text-decoration: underline;
  text-decoration-color: var(--accent-primary);
  text-decoration-thickness: 1px;
  text-underline-offset: 3px;
  transition: color 150ms, text-decoration-color 150ms;
}

a[href]:not([class]):not(nav a):hover {
  color: var(--accent-secondary);
  text-decoration-color: var(--accent-secondary);
}
```

This catches any inline links in paragraphs that:
1. Have an href (are real links)
2. Don't have a class (aren't styled components like buttons)
3. Aren't inside nav elements (navigation links are contextually distinct)

The transition timing is 150ms per research recommendation (not 300ms which feels sluggish).
